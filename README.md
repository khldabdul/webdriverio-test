# webdriverio-test

## How to Run

#### Install the dependencies

```
npm install
```

run your android emulator

to make sure Android Emulator is running, run this command line:

```
adb devices
```

#### Start your Appium Server

```
appium -p 4723 --plugins=images
```

#### Run your Test

```
npm run wdio
```
